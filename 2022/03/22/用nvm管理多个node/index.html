但是按照沿着路径找下去发现里面已经有这三行代码。因此不得不另辟蹊径。

直接选择降低node版本，直接使用nvm进行版本管理。

nvm的安装教程

准备工作

  • 首先我们需要卸载已经安装好的node（最好记住原先版本，之前项目可能都是这个版本）
  • 然后去安装node的文件夹中删除所有残余的文件，（把之前的环境也删了——我是删了）

正式安装

  1. 访问https://github.com/coreybutler/nvm-windows/releases
  2. 下载 nvm-setup.zip,解压之后会有个.exe安装程序，安装
  3. 找到安装目录下面的settings.txt文件直接追加到文件中
    node_mirror: https://npm.taobao.org/mirrors/node/
    npm_mirror: https://npm.taobao.org/mirrors/npm/
  4. 按上面这个步骤，系统会自动配好nvm环境。

基本操作

  1. 打开cmd，使用 nvm ls 命令查看本地已有版本
  2. 使用 安装命令 nvm install v16.14.0 即可（别忘了写v）
  3. 使用 安装命令 nvm install v12.13.0 即可
  4. 再次使用 nvm ls命令查看本地已有版本。发现16.14.0和12.13.0均存在
  5. 切换版本 nvm use 12.13.0（这里如果报错：nvm use报错 exit status 1: ‘ 出现乱码，多半是权限不够，用管理员权限打开cmd）

常用命令

  • nvm ls // 查看已安装node版本
  • nvm install vXX // 安装对应vXX版本的node
  • nvm uninstall vXX // 卸载对应vXX版本的node
  • nvm use xxx // 选择使用XXX版本
